To run a crawler that collect documents from the targeted website, you must follow the next steps in the following order:
  1. Create a template from here: Templates
  2. Create a crawler from here: Crawlers
  3. Run the crawler from here: Runners
Template represent the blue print or the sheet that will be collected from the internet, this varies from one use case to another. One example can be used is to collect the title and price of a product.
  1. Click on the button 'Create a template' to create a fresh template. Enter the name of the template and click on 'Create'.
  2. Create inspectors, inspectors are the elements on which we would like to inspect in our example we should like to create two inspectors 'Title' and 'Price'.
    • Expand the template by clicking on the '>' icon.
    • Click on the 'Create an inspector' button to create a new inspector.Enter the name as 'Title', the Selector XPath example: '//*[contains(@class, 'title')]' and the Attribute as the src for the images. Do the same for the 'Price'.
Before creating a crawler you have to create a template first.

Click on the button 'Create a crawler' to create a fresh crawler. Enter the required fields and click on 'Create'. All fields contain help-text describes their function.
After the crawler is completed and you are happy with the result, you can go to the indexing view.

Click on the button 'Create an indexer' to create a fresh index. Enter the required fields and click on 'Create'. All fields contain help-text describes their function.

Once the indexing configuration is saved you can start running it by clicking on 'Start indexing'.

Once the indexer is completed navigate to the searching view and pick up the indexer you like from the indexers drop menu.
